# Support

Thanks for using **agent-gpu**! Here's where to go depending on what you need.

## Questions & discussion

For usage questions, ideas, and general discussion, use
[GitHub Discussions](https://github.com/jaypetez/agent-gpu/discussions).

## Bugs & feature requests

Open an issue using the templates:

- [Bug report](https://github.com/jaypetez/agent-gpu/issues/new?template=bug_report.yml)
- [Feature request](https://github.com/jaypetez/agent-gpu/issues/new?template=feature_request.yml)

Search [existing issues](https://github.com/jaypetez/agent-gpu/issues) first — your
question may already be answered.

## Security vulnerabilities

**Do not** open a public issue for security problems. Follow the process in
[SECURITY.md](SECURITY.md) to report privately.

## Project status

agent-gpu is in early development; the roadmap is tracked on the
[project board](https://github.com/users/jaypetez/projects/10). Response times may vary.
See [CONTRIBUTING.md](CONTRIBUTING.md) if you'd like to help.
